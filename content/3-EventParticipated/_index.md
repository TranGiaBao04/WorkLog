# Events Participated

## Building Full-Stack Observability on AWS with Datadog

**Date & Time:** 27/02/2026 (14:00 – 17:00)  
**Location:** AWS Event Hall, Tầng 26, Bitexco Financial Tower, Q1, TP.HCM  
**Role:** Attendee  

**Summary:**  
A hands-on technical workshop co-hosted by Datadog and AWS, focusing on building full-stack observability in modern cloud systems. The session emphasized integrating metrics, logs, and traces into a unified monitoring platform, along with real-time infrastructure and application monitoring on AWS.

Participants worked in a sandbox environment to explore observability workflows, detect performance bottlenecks, and perform root cause analysis in distributed architectures.

**Outcomes:**  
- Hands-on experience with **full-stack observability implementation on AWS**  
- Learned how to correlate **metrics, logs, and traces** effectively  
- Practiced **troubleshooting and root cause analysis** in distributed systems  
- Gained insights into **production-grade monitoring strategies** using Datadog

## AWS re:Invent Recap HCMC

**Date & Time:** 04/02/2026  
**Location:** Ho Chi Minh City, Vietnam  
**Role:** Attendee  

**Summary:**  
A community-driven event summarizing the most important announcements and innovations from AWS re:Invent. The session covered new AWS services, architectural best practices, and real-world use cases across cloud-native applications, DevOps, and scalable system design.

Speakers also shared practical insights on optimizing cost, improving performance, and designing resilient cloud architectures based on the latest AWS advancements.

**Outcomes:**  
- Updated knowledge on **latest AWS services and innovations**  
- Gained insights into **modern cloud architecture patterns**  
- Learned strategies for **cost optimization and system scalability**  
- Expanded network within the **AWS developer and cloud community**
