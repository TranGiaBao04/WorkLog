This section summarizes my learning journey throughout the **first five weeks** of the **AWS FCJ Internship** program.

During this period, I gradually explored both foundational and more advanced concepts in **Cloud Computing**, while also developing a stronger system mindset and gaining hands-on experience with practical deployment scenarios on AWS.

Beyond learning AWS core services, the program also helped me strengthen my skills in **architecture thinking, system operations, and real-world problem solving** — all of which are essential for roles such as **Cloud Engineer** or **Solutions Architect**.

---

## Learning Roadmap for the First 5 Weeks

### **Week 1: AWS Fundamentals – Core Foundations**

- Learning how AWS works and understanding key cloud concepts
- Creating and securing an AWS account with MFA and IAM best practices
- Practicing with **IAM Users, Groups, and Roles**
- Configuring **AWS Budgets** to monitor and control costs
- Becoming familiar with the **AWS Console**

---

### **Week 2: IAM & Networking**

- Practicing **IAM** in depth, including Users, Groups, Policies, Roles, and Switch Role
- Understanding the architecture of **VPC**
- Comparing **Security Groups** and **Network ACLs**
- Deploying and connecting to an **EC2 instance**
- Configuring networking components such as:
  - **Hybrid DNS (Route 53 Resolver)**
  - **VPC Peering**
  - **AWS Transit Gateway**

---

### **Week 3: Core AWS Services & Project Direction**

- Developing the idea and architecture for the **final project**
- Practicing key services:
  - **EC2** (deployment, configuration, and custom AMIs)
  - **S3** (static website hosting and versioning)
  - **IAM Role**, **Cloud9**
- Working with **RDS** to build the database layer
- Reviewing and reinforcing previously learned knowledge

---

### **Week 4: System Design & Optimization with the Well-Architected Framework**

- Applying AWS’s five pillars:
  - **Operational Excellence**
  - **Security**
  - **Reliability**
  - **Performance Efficiency**
  - **Cost Optimization**

- Practicing with:
  - Automation using Lambda
  - Monitoring with CloudWatch
  - CI/CD with CodePipeline
  - Container deployment using Docker and ECS

---

### **Week 5: Project Preparation & Environment Setup**

- Working with the team to define project scope and direction
- Designing the overall system architecture
- Preparing a suitable development environment
- Experimenting with AWS services needed for the project
- Attending the **AWS re:Invent Recap HCMC event (04/02/2026)**
- Reviewing knowledge from Weeks 1 through 4

---

> **Note:**  
> This section is a summary of the **first five weeks** of the **12-week AWS FCJ Internship** program.

> The following weeks will focus more on **real project implementation, system integration, and production operations**.
