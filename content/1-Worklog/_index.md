In this section, I document my learning journey during the first **5 weeks** of participating in the **AWS FCJ Internship Program**.

Throughout this period, I was able to explore and practice **Cloud Computing** concepts from foundational knowledge to more advanced system design and operations.

The program not only helped me understand core AWS services, but also developed my ability in **system thinking, hands-on implementation, and real-world problem solving** — essential skills for roles such as **Cloud Engineer** or **Solutions Architect**.

---

## First 5-week learning journey

### **Week 1: AWS Fundamentals – Foundation Setup**

- Learned core AWS concepts and cloud deployment models
- Created and secured AWS account (MFA, IAM best practices)
- Set up **IAM Users, Groups, and Roles**
- Configured **AWS Budgets** for cost control
- Explored AWS Console and basic services

---

### **Week 2: IAM & Networking Fundamentals**

- Practiced **IAM** (Users, Groups, Policies, Roles, Switch Role)
- Understood **VPC architecture** and networking basics
- Compared **Security Groups vs Network ACLs**
- Launched and connected to **EC2 instances**
- Implemented:
  - **Hybrid DNS (Route 53 Resolver)**
  - **VPC Peering**
  - **AWS Transit Gateway**

---

### **Week 3: Core AWS Services & Project Planning**

- Developed ideas and architecture for the **final project**
- Worked with:
  - **EC2** (deployment, configuration, custom AMI)
  - **S3** (static website hosting, versioning)
  - **IAM Roles** and **Cloud9**
- Built and managed databases using **RDS**
- Reinforced knowledge from previous weeks

---

### **Week 4: System Design & Optimization (Well-Architected)**

- Applied AWS Well-Architected pillars:
  - **Operational Excellence** (Lambda, Systems Manager, automation)
  - **Security** (IAM Boundaries, WAF, Security Hub)
  - **Reliability** (AWS Backup, VPC Peering, Transit Gateway)
  - **Performance** (Docker, ECS, CI/CD with CodePipeline)
  - **Cost Optimization** (Savings Plans, right-sizing EC2)

---

### **Week 5: Project Preparation & Environment Setup**

- Collaborated with team members to define project scope
- Designed system architecture and selected AWS services
- Set up development environment for project implementation
- Explored AWS services for:
  - Cost optimization
  - Performance improvement
- Attended **AWS re:Invent Recap HCMC (04/02/2026)**
- Reviewed and consolidated knowledge from Weeks 1–4

---

> **Note:**
> This is a summary of the first **5 weeks** out of the **12-week AWS FCJ Internship Program**.

> The following weeks will focus on **project implementation, deployment, and real-world system integration**.
