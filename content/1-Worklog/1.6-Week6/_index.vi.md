### Mục tiêu Tuần 6

Tuần này tập trung vào việc tìm hiểu và thực hành với cơ sở dữ liệu quan hệ trên SQL Server, bao gồm thiết kế bảng, thao tác dữ liệu và đánh giá hiệu năng truy vấn.

- **Tìm hiểu nền tảng SQL Server**:
  - Database, tables, rows, columns
  - Primary key, foreign key
  - Schema và cách thiết kế bảng

- **Tạo và cấu hình cơ sở dữ liệu**:
  - Tạo database và các bảng dữ liệu
  - Thiết lập các ràng buộc cần thiết (constraints)

- **Thực hành CRUD với SQL**:
  - INSERT
  - SELECT
  - UPDATE
  - DELETE

- **Sử dụng công cụ hỗ trợ**:
  - SQL Server Management Studio (SSMS) / Azure Data Studio
  - AWS RDS (SQL Server) hoặc môi trường local

- **Đánh giá hiệu năng**:
  - Theo dõi thời gian truy vấn (query time)
  - Quan sát execution plan ở mức cơ bản
