### Week 1 Objectives

During the first week, the main focus was on getting familiar with AWS, setting up the initial environment, and building a foundation in account management as well as cost control. The key objectives included:

- **Getting an overview of AWS**:
  - Understanding how AWS operates and the cloud computing model.
  - Learning about core services such as EC2, S3, IAM, VPC, ...

  - **Teamwork**:
  - Forming a study/working group.
  - Connecting, getting acquainted, and interacting with team members.

- **Getting familiar with the lab environment**:
  - Understanding the structure of the lab exercises.
  - Becoming familiar with the process of deploying and completing labs.

- **Setting up an AWS account**:
  - Creating an AWS account under the Free Tier plan.
  - Enabling MFA for the Root User.
  - Creating an IAM User with administrative privileges.
  - Creating a Group and assigning Administrator permissions.
  - Becoming familiar with the AWS Console interface.

- **Managing costs with AWS Budgets**:
  - Setting up a Cost Budget.
  - Setting up a Usage Budget.
  - Setting up a Reservation Budget.
  - Setting up a Savings Plans Budget.
  - Configuring cost alerts.
  - Cleaning up resources after use.

### Week 1 Achievements

#### What was accomplished

- Gained a clearer understanding of the services provided by **AWS**
- Formed a group, made friends, and connected with many new people
- Clearly understood the details and deployment process of each **lab exercise**
- Successfully created an AWS account with **Free Tier (~$100 credit)**
- Enabled **MFA for the Root User** to improve security
- Created a separate **IAM User** to avoid using the Root Account
- Became familiar with the **AWS Console**
- Set up monthly budgets and cost alerts
- Configured **cost alerts** and cleaned up resources

#### Best Practices Summary

- **Security**:
  - Avoid using Root Account daily
  - Always enable MFA

- **Cost Management**:
  - Track usage with AWS Budgets
  - Set alerts for thresholds

- **Learning**:
  - Hands-on labs
  - Team collaboration
