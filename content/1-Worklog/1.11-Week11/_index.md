### Week 11 Objectives

This week, the team focused on the final preparation stage before deployment by running full integration testing, improving performance, validating the infrastructure, and strengthening security for the production environment. The main objective was to ensure that the system was stable, secure, and ready for official release.

- **Integration testing**:
  - Performing end-to-end testing across the entire system
  - Verifying data flow between modules
  - Detecting and resolving issues found during integration

- **Performance optimization**:
  - Refining database queries and backend processing logic
  - Improving system response times
  - Reducing infrastructure resource usage during operation

- **Infrastructure validation**:
  - Reviewing the configuration of EC2, RDS, and S3
  - Ensuring scalability and operational stability
  - Testing failover mechanisms to confirm availability

- **Production hardening**:
  - Applying security best practices
  - Securing environment variables and secrets configuration
  - Strengthening logging and access control mechanisms
