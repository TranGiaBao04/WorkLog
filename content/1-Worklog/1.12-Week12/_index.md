### Week 12 Objectives

This week focused on deploying the system to the production environment, completing the monitoring setup, and preparing for official operations. The main priority was to ensure that the platform could run reliably in a real-world environment, remain observable, and be ready for handover.

- **Deploying the system to production**:
  - Releasing the full system to the live environment
  - Configuring the required cloud services and domain settings
  - Ensuring stable operation after deployment

- **Setting up monitoring**:
  - Implementing monitoring and alerting mechanisms
  - Tracking logs and system performance
  - Building dashboards to observe the overall platform

- **Preparing for live operations**:
  - Performing a final overall system check
  - Completing handover documentation
  - Getting the system ready for real users
